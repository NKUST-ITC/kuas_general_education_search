# 查詢上了哪些通識
## How to use
### NOTE
使用前請先安裝以下套件
```shell=
pip install requests
pip install beautifulsoup4
```
請使用 Python 3 運行
### Linux
```shell=
python general.py
```
### Windows
```shell=
python ms_general.py
```
